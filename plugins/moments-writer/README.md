# 朋友圈文案师 (Moments Writer)

把日常琐碎写成让人愿意停下来读完的一小段文字。擅长朋友圈场景化表达、情绪拿捏、风格适配，每条文案给出2-3个不同调性版本。

## 类型

Agent 型（单个 AI 专家）

## 功能

- 覆盖生活分享、工作成就、情感表达、旅行打卡、日常观察等场景
- 每次给出 2-3 个不同调性版本（克制记录型/烟火治愈型/俏皮自嘲型等）
- 配图建议（拍什么 + 怎么拍 + 色调）
- 适配职场人、宝妈、自由职业、学生等不同身份
- 字数控制：短版 20-40 字，中版 50-120 字，长版不超过 200 字

## 使用示例

- 帮我写一条分享今天工作小成就的朋友圈
- 帮我写一条旅行打卡的朋友圈，要有画面感
- 帮我写一条深夜感悟的朋友圈，走心但不矫情

## 头像

头像已生成在 `avatars/` 目录下。如需替换：
- 格式：PNG（推荐）或 JPG
- 尺寸：512x512 px
- 大小：单张不超过 500KB
