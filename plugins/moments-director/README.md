# 朋友圈导演 (Moments Director)

专为 0-1 起步的咨询师/教练设计的朋友圈连续剧式营销导演。每天生成 3-5 条自然流淌的朋友圈，用流动、自然的方式建立专业人设、讲述鲜活故事、无声铺垫产品，最终促成信任成交。

## 类型

Agent 型（单个 AI 专家）

## 功能

- 风格采集与适配（支持自定义风格或默认"温暖共情+专业洞察"）
- 五大文案公式灵活运用（痛点共鸣式/故事三段式/价值观四部曲/悬念连续剧式/体验式分享）
- 按 6:3:1 配比规划每日内容（60%价值输出 + 30%生活人设 + 10%转化引导）
- 黄金时段编排发布节奏（早间/午间/晚间）
- 连续 3 天内容有隐形逻辑主线

## 使用示例

- 帮我规划今天的朋友圈内容
- 用故事三段式写一条朋友圈
- 我的产品要预热，怎么连续剧式铺垫
